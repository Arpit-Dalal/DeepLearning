# Game 2048 - Reinforcement Learning

This project demonstrates how to play the popular game *2048* using a *Reinforcement Learning (RL)* model. It leverages *Artificial Neural Networks (ANNs)* with *TensorFlow* and a customized *Gym environment* for training and testing.

## Introduction

2048 is a sliding tile puzzle where the objective is to combine tiles with the same value to reach the target value of 2048. This project uses Reinforcement Learning to train an AI agent capable of playing the game intelligently by learning optimal strategies.

The RL agent is trained using an Artificial Neural Network (ANN) implemented with TensorFlow, and a Gym environment is used for simulation and testing.

## Features
- Fully functional 2048 game environment.
- RL agent trained using *Deep Q-Learning* (DQN) or other RL algorithms.
- Integration with TensorFlow for building and training neural networks.
- Easy-to-use custom Gym environment for training and evaluation.
- Configurable parameters for training, such as learning rate, batch size, and exploration rate.
